# flxtheme

A centralized design token theming system for modern dashboards and web applications. Drop in a CSS file with your brand colors, typography, and border radius, and keep your styles consistent everywhere.

## Links

* 🌐 **Preview:** https://flxtreme.github.io/flxtheme/
* 📚 **Docs:** https://flxtreme.github.io/flxtheme
* 📦 **npm:** https://www.npmjs.com/package/flxtheme
* 💻 **GitHub:** https://github.com/flxtreme/flxtheme

## Installation

```bash
npm install flxtheme@latest
```

## Setup

```bash
npx flxtheme@latest init
```
